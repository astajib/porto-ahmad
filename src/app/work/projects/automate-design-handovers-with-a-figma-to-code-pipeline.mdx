## Overview

In this project, I focused on customization sales module and database migration from Odoo v.16 to Odoo v.17.

## Common Customizations

- **Quotation & Sales Order Templates**: Addi custom fields and modify layout.
- **Product and Pricing Rules**: Customize product configuration rules.
- **Access Control*: Customize user roles and field-level permissions.
- **Database Migrations**: Upgrade Odoo core and migration database from v.16 to v.17

## Technologies Used

- **Odoo ERP**
- **Python Programming**
- **Postgre Database**
- **Ubuntu Server OS**
